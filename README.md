# ResourceManagement

React Application with lightweight Node.js and Express Back-End <br/> 

### To Demo <br/>
* BackEnd (http://localhost:5000): <br/>
  * cd BackEnd <br/>
    * npm start <br/>
* FrontEnd (http://localhost:3000): <br/>
  * cd FrontEnd <br/>
    * npm start <br/>
